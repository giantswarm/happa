# Colors

## Our main colors
These are all the colors that we currently support and use for new code.
These colors should not be directly used via <code>COLORS</code> constant.
The preferred way is by using semantic color names from a Grommet's theme entry.
To reference a color in styled components, <code>normalizeColor</code> utility function from Grommet must be used,
e.g <code>normalizeColor('background-front', theme)</code>.
